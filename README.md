🚗 Car Rent Management System

A complete desktop-based Car Rental Management System developed 
using Java Swing and MySQL (JDBC). Built as part of SW121 – 
Object Oriented Programming course at Mehran University of 
Engineering and Technology (MUET), Khairpur.

🛠️ Technologies Used
- Language     : Java (JDK 17+)
- GUI          : Java Swing (Custom Painted Components)
- Database     : MySQL 8.0
- Connectivity : JDBC (mysql-connector-j-9.6.0)
- Architecture : MVC (Model-View-Controller)
- IDE          : IntelliJ IDEA

✨ Features
- Role-based login (Admin / Customer)
- 5 Car categories: Economy, Standard, Luxury, SUV, Van
- Car Fleet Management (Add/Edit/Delete/Search)
- Driver Management with experience tracking
- Car Rental Booking with/without driver
- Auto fare calculation (Days × Rate/Day)
- Return & Cancel rental
- Print receipts (Java Print API)
- Admin Dashboard with live statistics
- Customer Portal with personal rental history
- Search functionality in every panel

🗄️ Database
- 23 pre-loaded cars
- 10 active drivers
- 4 sample users
- 3 sample rentals
